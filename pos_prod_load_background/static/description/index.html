Odoo 10 Point of Sale Extension


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Loading products and customers in background to improve speed of Point of Sale front-end.

  • When user start POS session at that time all products and customers are load into background.
  • We can search partners while partners are loading.
  • when products are loading at that time we can search products via barcode scanner or type manually on search products box.
  • Also we can synchronized products which is updated into backend after load products into pos.
